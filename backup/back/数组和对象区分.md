## 判断2个数组是否相等

说明：Array数组为引用类型，所以哪怕 []===[] 都会返回false，所以可通过数组toString()方法返回的字符串判断是否相等。

示例：

	console.log([]===[]); // => false
	console.log(['a', 'b'] === ['a', 'b']); // => false
	console.log(['a', 'b'].toString() === ['a', 'b'].toString()); // true
